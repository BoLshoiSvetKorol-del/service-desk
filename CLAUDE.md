# Service Desk System

Корпоративная система управления заявками и инцидентами для службы поддержки.

## Технологический стек

### Backend
- **Python 3.12** — основной язык
- **FastAPI** — веб-фреймворк с автоматической генерацией OpenAPI/Swagger документации
- **PostgreSQL** — реляционная база данных
- **SQLAlchemy** — ORM для работы с БД
- **Alembic** — система миграций БД
- **Celery** — фоновые задачи (email-уведомления, SLA-мониторинг)
- **Redis** — брокер сообщений для Celery + кеш
- **WebSocket/SSE** — real-time уведомления

### Frontend
- **React 18** — UI библиотека
- **TypeScript** — типизированный JavaScript
- **Ant Design (antd)** — UI компоненты
- **React Router** — маршрутизация
- **Axios** — HTTP клиент
- **WebSocket Client** — real-time обновления

### Аутентификация
- **JWT токены** — stateless аутентификация
- Access token (15 мин) + Refresh token (7 дней)

## Функциональность MVP

### Роли пользователей
- **Администратор** — полный доступ ко всем заявкам и настройкам
- **Агент поддержки** — доступ к заявкам своего отдела и назначенным на него
- **Пользователь** — создание и просмотр только своих заявок

### Управление заявками
- Создание заявок через веб-форму
- Уникальный номер формата `SD-2026-00001`
- Типы заявок (справочник): вид услуги, направление работ
- Приоритеты: низкий, нормальный, высокий, критичный
- Автоматический расчёт SLA по приоритету
- Назначение на отдел → исполнителя
- Автоматическая маршрутизация по типу заявки

### Жизненный цикл заявки
Статусы: `новая` → `в работе` → `ожидает информации` → `выполнена` / `отменена`

### SLA и эскалация
- Расчёт в рабочих часах (пн-пт 9-18)
- Дедлайны по приоритетам:
  - Низкий: 24 раб.ч
  - Нормальный: 8 раб.ч
  - Высокий: 4 раб.ч
  - Критичный: 1 раб.ч
- Автоматическая эскалация при приближении дедлайна
- Email-уведомления через Celery

### Коммуникация
- Комментарии к заявкам
- Прикрепление файлов (до 10 МБ)
- Real-time уведомления о изменениях
- История всех действий по заявке

### Поиск и отчёты
- Поиск по номеру, клиенту, дате, статусу, приоритету, исполнителю
- Сохранение фильтров
- Отчёты: количество заявок, распределение по статусам, среднее время обработки
- Экспорт в Excel/CSV
